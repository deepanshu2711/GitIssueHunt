## ⚙️ Setup Instructions

1. **Fork the repository**

2. **Clone your fork**

   ```bash
   git clone https://github.com/YOUR_USERNAME/GitIssueHunt.git
   cd GitIssueHunt

3. npm install

4. npm run dev
